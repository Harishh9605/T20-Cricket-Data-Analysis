# T20 Cricket Data Analysis

## Project Overview
This project analyzes T20 cricket batting performance using SQL, Python, and Excel.

## Tools Used
- MySQL: Data storage and analysis
- Python (pandas, matplotlib): Data extraction, analysis, and visualization
- Excel: Final reporting

## Workflow
1. Raw CSV data loaded into MySQL
2. SQL queries used to analyze player performance
3. Python connects to MySQL and automates analysis
4. Results exported to Excel
5. Visualizations created using Python

## Key Analysis
- Top 10 run scorers
- Top 10 batting averages
- Top 10 strike rates
- Relationship between average and strike rate

## Output
- Excel files stored in the `output/` folder
- Visual charts generated using matplotlib

## Skills Demonstrated
- SQL querying and aggregation
- Python automation
- Data visualization
- End-to-end data analytics workflow
